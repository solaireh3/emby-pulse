Emby 核心连接

系统底层连接配置。如果 EmbyPulse 与 Emby 部署在同一台机器或同一个 Docker 网络下，建议填写内网 IP 提高通信效率。

例：http://172.17.0.1:8096 或 http://192.168.31.2:8096

在 Emby 控制台 -> 高级 -> API 密钥中生成，需管理员权限。

下载自动化

连接你的 MoviePilot。配置后，当你在审批中心点击“通过”时，系统会自动将资源推送到 MP 进行下载。
